Flashing Guide

Flashing takes about two minutes with a USB cable — no special hardware required.

Before you start, you will need: a free ThingsBoard account, a DC 5V 1A power supply for the board, and a USB cable.

  1. Download the firmware

    Get the latest .bin file from the downloads section above.

  2. Connect the board

    Plug the relay board into your computer with a USB cable.

  3. Flash the firmware

    In Chrome or Edge, open the Espressif Web Flasher, click Connect, choose the .bin file, set the flash address to 0x0, and click Program. Or use the command line:

    pip install esptool
esptool.py --chip esp32 --baud 921600 write_flash 0x0 smart_relay_v1.0.bin

    The firmware file is a complete flash image (bootloader + partition table + application), so it is always written to address 0x0.

  4. Restart the board

    Power-cycle the board after flashing completes. You're ready to configure it.
