Prerequisites

  • ThingsBoard account — sign up free at thingsboard.cloud to configure and control the board
  • DC 5V 1A power supply for the relay board

How to flash the firmware (ESP32)

  1. Download the latest .bin firmware file above.
  2. Connect the relay board to your computer with a USB cable.
  3. Flash in your browser (easiest, Chrome or Edge): open Espressif Web Flasher, click Connect and select the board's serial port, choose the downloaded .bin file, set Flash Address to 0x0, then click Program.
  4. Or flash with esptool (command line, requires Python): 
    pip install esptool
esptool.py --chip esp32 --baud 921600 write_flash 0x0 smart_relay_v1.0.bin
  5. Power-cycle the board after flashing completes.

The firmware file is a complete flash image (bootloader + partition table + application), so it must always be written to address 0x0.
